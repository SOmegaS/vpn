Использование

Работоспособность проверена только на Linux. На большинстве протестированных нами дистрибутивах (это Manjaro,Linux Mint и Ubuntu) достаточно просто использовать готовый файл.

Использование готового файла

  • Скачать исполняемый файл
  • Зайти в консоль, перейти в папку загрузок
  • chmod +x vpn
  • sudo ./vpn
  • Следовать промптам в консоли
  • Если ваш NAT - Symmetric, то программа не может работать в режиме listen, только в режиме connect. Определить тип NAT

Самостоятельная сборка

  • Удостоверьтесь, что у вас стоит Go.
  • Склонируйте репозиторий
  • CGO_ENABLED=1 GOOS=linux GOARCH=amd64 CC=x86_64-linux-gnu-gcc go build -o vpn ~/cmd/client/main.go
